# eleven_rag

一个基于 Java 21、Spring Boot 3.3、Spring Cloud Alibaba 与 Spring AI Alibaba 的 RAG 知识库项目骨架。

## 技术栈

- Java 21
- Spring Boot 3.3.0
- Spring Cloud 2023.0.3
- Spring Cloud Alibaba 2023.0.3.4
- Spring AI Alibaba 1.0.0.4
- Spring Web
- Spring Validation

## 当前初始化内容

- 标准 Maven 单模块工程
- 基础启动类
- `/api/health` 健康检查接口
- `application.yml` 基础配置
- `.gitignore`

- 接入文档上传与解析
- 增加 Embedding / 检索 / RAG 编排模块
- 接入向量存储与对象存储
